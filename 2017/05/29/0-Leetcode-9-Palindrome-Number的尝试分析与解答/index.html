闲话说的太多。这次的#9，是要判断一个数是不是回文数，也就是说数字是不是回文的样子。要求是不能用额外的存储空间。

如果你从数学上来分析回文数的话，可以有两个描述

  1. 把这个数的每一个数字前后反转一下，应该会得到一样的数字。
  2. 如果我们把左手食指指向回文数最左边的数位，右手食指指向最右边的数位，两个手指指向的数是一样的，左右手相互靠拢一位，数字也还是一样的。

但是这是数学描述，如果以计算机视角来说会遇到几个问题：

  1. 怎么样得到这个数的每一位数字？
  2. 怎样遍历每一个数位？
